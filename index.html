What is CI/CD?

CI/CD is a set of practices that helps developers deliver changes faster and more safely. Continuous Integration (CI) automatically checks code after each push to make sure it is valid. Continuous Deployment/Delivery (CD) automatically publishes the latest version when checks pass. This reduces manual work, catches mistakes early, and keeps software always up to date.

My Pipeline

When I push to the main branch, GitHub Actions runs a workflow automatically. It checks that the required files exist, then uploads the website files and deploys them to GitHub Pages. After the workflow succeeds, the live website updates with my latest changes.
